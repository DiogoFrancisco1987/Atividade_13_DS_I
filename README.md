# 🚀 Portal de Entretenimento 

Bem-vindo ao meu projeto da **Agenda 13** do curso de Desenvolvimento de Sistemas (ETEC). Este portal foi criado para praticar conceitos avançados de HTML5 e CSS3, com foco em semântica e layouts responsivos.

## 🛠️ Tecnologias Utilizadas
*   **HTML5** (Tags semânticas)
*   **CSS3** (Flexbox, IDs, Classes e Seletores)

## 📋 Funcionalidades
- [x] Layout em 3 colunas usando Flexbox.
- [x] Estilização em três níveis (Inline, Interno e Externo).
